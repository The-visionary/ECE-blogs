Computer Networking


Rounak Saha

In simple words, if someone needs to define what computer networking is we can say it is the name given to the full scope of how computers communicate with each other. So, a question might arise that how the computers communicate with each other? Are there some specific rules which it needs to follow? Your guess is correct. A definite set of rules that computer must follow in order to communicate properly is called protocol.
There are basically two models of Computer Networking -

  • TCP/IP Five Layer Network Model
  • OSI(Open System Interconnection) Seven Layer Model

TCP/IP Five-Layer Network Model

  1. Physical Layer - It represents physical devices that interconnect computers. It follows the protocol 10BaseT,802.11.
  2. Data Link Layer -It is responsible for communication of network devices by interpreting signals. This layer follows the protocol Ethernet/Wi-Fi with protocol data unit as Frames and MAC Address addressing.
  3. Network Layer - It allows different networks to communicate with each other through devices known as routers. This layer follows the protocol IP with protocol data unit as Datagram.
  4. Transport Layerr - It sorts out client and server programs are supposed to get that data. It follows TCP/UDP protocol with protocol data unit as segment following Port #’s addressing.
  5. Application Layer - It follows HTTP,SMTP protocol and messages as protocol data unit.
OSI Seven Layer Model is now not used as much as it is outdated but it must be noted that here only application later is divided into three more sub layers.

Basic Networking Devices

  • Cables - It connects different devices to each other, allowing data to be transmitted over them.
  • Hub - It is a physical layer device that allows communications from multiple computers at once.
  • Router - It is a device which helps us to forward data between independent networks.

Duplex & Simplex Communication

Duplex Communication is the concept that information can flow in both directions across the cable whereas in simplex communication the process is unidirectional. Client computer which only sends data to the server is Simplex Communication. Client Computer which sends and receives data simultaneously is Full Duplex whereas client computer which takes turn to send or receive data is Half Duplex.

Some Important Terms & Concepts of Networking

  • Ethernet - It is primarily a standard communication protocol used to create local area networks.
  • MAC Address - Also known as Media Access Control Address, is a globally unique identifier attached to an individual network interface.
  • Transmission Control Protocol (TCP) - It is a protocol that establishes a connection between client and server so that data can be sent.
  • TCP Control Flags
    • URG (Urgent)-It indicates the segment is considered urgent and pointer field has more data about this.
    • ACK (Acknowledged)-It means that the acknowledgement number field should be examined.
    • PSH (Push)-The transmitting device wants to push data to the receiving application immediately.
    • RST (Reset)-It is used to indicate that the sender will not accept any data.
    • SYN (Synchronize)-It is used when first establishing a TCP connection and makes sure the receiving end knows to examine the sequence number field.
    • FIN (Finish)-When it is set to one, it means transmitting computer doesn’t have any data to send and communication can be closed.
  • Three Way Handshake - Handshake is a way for two devices to ensure that they are speaking the same protocol and will be able to understand each other. If there are two computers Computer A and Computer B, Computer A sends SYN (Synchronize) flag to Computer B which it receives and sends SYN/ACK flag to Computer A and then again Computer A sends ACK (Acknowledged) flag to Computer B completing the three way handshake.
  • Firewall - A device which blocks traffic by meeting certain criteria.
  • Domain Name System (DNS) - A global and highly distributed network service that resolves strings of letters into IP addresses for us.
  • Virtual Private Network (VPN) - A technology that allows for the extension of a private network to hosts that are not present on the local network.
  • Wired Equivalent Privacy (WEP) - It is an encryption technology which only uses 40bits for its encryption keys. WEP was quickly replaced by WPA or Wi-Fi Protected Access. WPA uses 128-bit key. Today the most commonly used encryption algorithm for wireless network is WPA2, which uses a 256-bit key.

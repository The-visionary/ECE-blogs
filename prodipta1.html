The concept of the ESP8266 wi-fi Module


- Prodipta Pal

ECE





We all are fewer concepts about the wi-fi module, web server, and network system. We not only know how it works or where it works but also the application of this module in IoT. So here, I will discuss this such of things like the ESP8266 wi-fi Module.

ESP8266 :

The ESP8266 is a low-cost wi-fi microchip, with a full TCP/IP stack and microcontroller capability, produced by Espressif systems in Shanghai, China. This small module allows microcontrollers to connect to a wi-fi network and make simple TCP/IP connections using Hayesstyle commands.



ESP8266-01 Features:

1. Low cost, compact and powerful wi-fi Module

2. Power Supply: +3.3V only

3. Current Consumption: 100mA

4. I/O Voltage: 3.6V (max)

5. I/O source current: 12mA (max)

6. Built-in low power 32-bit MCU @ 80MHz

7. 512kB Flash Memory

8. Can be used as Station or Access Point or both combined

9. Supports Deep sleep (<10uA)

10. Supports serial communication hence compatible with many development platforms like Arduino

11. Can be programmed using Arduino IDE or AT-commands or Lua Script

HField of uses of ESP8266:

The module can work both as an Access point (can create hotspot) and as a station (can connect to wi-fi); hence it can easily fetch data and upload it to the internet, making the Internet of Things (IoT) as easy as possible. Another exciting feature of this module is that it can be programmed using the Arduino IDE, which makes it a lot more user friendly. However, this version of the module has only 2 GPIO pins, so we have to use it another microcontroller like Arduino else we can look on to the more standalone ESP-12 or ESP-32 version.

Working Principle of ESP8266:

The ESP8266 module works with 3.3V only; anything more than 3.7V would damage the module; hence be cautious with our circuits. The best way to program an ESP-01 is by using the FTDI board that supports 3.3V programming. If we donâ€™t have one, it is recommended to buy one, or for the time being, we can also use an Arduino board. One common problem that everyone faces with ESP-01 is powering up the problem. The module is a bit power hungry while programming, and hence we can power it with a 3.3V pin on Arduino or use a potential divider. So, it is important to make a small voltage regulator for 3.31v that could supply a minimum of 500mA. One recommended regulator is the LM317, which could handle the job easily.
